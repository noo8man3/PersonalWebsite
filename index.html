>Projects_

Denial-of-Service Client - Java, JavaFX

This Java program spams UDP packets at a user-designated IP address and port number. JavaFX is used heavily to deliver easily readable analytics and provides a crisp, clean user interface.

Password Cracker - Java, JavaFX

This Java program performs a dictionary attack on a hashed password file. JavaFX is used to simplify the process of selecting text files and initializing the attack once said files are selected.

Have I Been Owned? - HTML, CSS, JavaScript

This webpage is a one-stop shop to see if your credentials are compromised. The idea is very simple: If you willingly enter your credentials on this site, the webpage tells you to change them.

Budget App - HTML, CSS, JavaScript

Deadlines - Java, XML

This Android application helps organize any events in your life in just a couple taps. Deadlines offers a stellar user interface and ability to configure the app to what's important to you.

Password Generator - Python
